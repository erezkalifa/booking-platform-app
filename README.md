# Booking Platform App 🏠

A modern, responsive property booking platform built with React. This application allows users to browse, filter, and save their favorite properties, featuring an intuitive user interface and smooth user experience.

## 🌟 Features

- **Property Browsing**: Browse through a curated list of properties with detailed information
- **Advanced Filtering**: Filter properties by:
  - Location
  - Check-in/Check-out dates
  - Number of bedrooms
  - Number of guests
  - Price range
- **Image Gallery**: View property images in an elegant gallery with:
  - Main large image display
  - Thumbnail previews
  - Full-screen gallery view
- **Wishlist**: Save and manage favorite properties
- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean and modern interface with smooth animations and transitions

## 🚀 Getting Started

### Prerequisites

- Node.js (v14.0.0 or higher)
- npm (v6.0.0 or higher)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/booking-platform-app.git
```

2. Navigate to the project directory:

```bash
cd booking-platform-app
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

The application will open in your default browser at `http://localhost:5173`
